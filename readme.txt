== Description ==

This plugin uses '[Contact Form 7](https://ja.wordpress.org/plugins/contact-form-7/)' to register information in the target database of '[WiLL Mail](https://willcloud.jp/)'.

= Features =

* This plugin is for creating '[WiLL Mail](https://willcloud.jp/)' input form using '[Contact Form 7](https://ja.wordpress.org/plugins/contact-form-7/)'.
* This plugin does not work without '[Contact Form 7](https://ja.wordpress.org/plugins/contact-form-7/)'.
* You can send any information conceived with '[WiLL Mail](https://willcloud.jp/)'.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-willmail-put` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Please open the setting page and register the keys on the API page of '[WiLL Mail](https://willcloud.jp/)'.

== Frequently Asked Questions ==

= What is the version of the WiLL Mail API supported? =

It corresponds to API version 1.0.0 and Document version 9.

= Will an information be sent to 'WiLL Mail' if an error occurs in 'Contact Form 7' input check? =

No, it will not send it.
